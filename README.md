# Professional Architecture Portfolio

A premium, modern portfolio website for Mariam Belkacem - Holistic Architect, Interior Design & Spatial Well-Being Specialist.

## Features

- **Responsive Design**: Fully responsive across all devices
- **Modern UI**: Dark theme with gold accents and glassmorphism effects
- **Smooth Animations**: Scroll-triggered animations and hover effects
- **Professional Sections**:
  - Hero with contact information
  - Professional Profile
  - Core Competencies
  - Professional Experience
  - Education
  - Certificates & Professional Training
  - Technical Skills with progress bars
  - Professional Approach
  - Works Showcase with modal lightbox

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Playfair Display, Inter)

## Deployment on Render

### Option 1: Static Site (Recommended)

1. Go to [Render Dashboard](https://dashboard.render.com/)
2. Click **"New +"** → **"Static Site"**
3. Connect your GitHub repository: `zakariabouchelaghm/Portfolio-archi`
4. Configure:
   - **Name**: `portfolio-archi` (or your preferred name)
   - **Branch**: `main`
   - **Build Command**: Leave empty (no build needed)
   - **Publish Directory**: `.` (root directory)
5. Click **"Create Static Site"**

Your site will be live at: `https://portfolio-archi.onrender.com` (or your custom name)

### Option 2: Manual Deploy

1. Download the repository as ZIP
2. Go to Render Dashboard
3. Create a new Static Site
4. Upload the files manually

## Local Development

Simply open `index.html` in your browser. No build process required!
